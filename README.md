## Інструкції по запуску

1. **Клонування репозиторію**
   ```bash
   git clone <URL репозиторію>
   cd назва-проекту

2. **Встановлення залежностей**

    npm install

3. **Запуск застосунку**

    npm start

4. **Перегляд у браузері**

    Відкрийте ваш браузер і перейдіть за адресою http://localhost:3000
